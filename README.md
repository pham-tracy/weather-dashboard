# weather-dashboard

## Description

This application allows users to view the daily and five-day weather forecast for any city they search for. Having this weather forecast helps users plan accordingly.

This application was created with HTML, CSS, and JavaScript. Specific technologies include using Fetch API to fetch weather data from OpenWeather (https://openweathermap.org/).
